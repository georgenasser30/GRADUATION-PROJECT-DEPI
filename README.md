Huawei Network Administrator Graduation Project – DEPI (Round 3)
Project Overview

This repository contains the full graduation project Presentation and documentation for the Huawei Network Administrator Track – DEPI (Round 3), part of DEPI.
The project simulates a complete enterprise-level network for a three-floor oil company, designed with high availability, strong security, and real-world enterprise standards.

Project Components
1. Network Architecture

Full Core – Distribution – Access topology

Access, Distribution, and Core switches with ETH-Trunk links

Redundant connections to two service providers

RSTP and Edge-Port features for fast recovery

VRRP for gateway high availability

Centralized and secure DHCP

Wireless setup using AC + APs with CAPWAP

2. Security Implementation

Next-Generation Firewall (NGFW) with detailed policies and routing

IDS simulation using Port Mirroring (SPAN)

Switch-level protections: Anti-ARP, DHCP Snooping, Port Security Groups

Layered security design for realistic enterprise scenarios

3. Web Platform

Full Front-End + Back-End website to represent company structure, services, and management workflows

Network documentation, diagrams, and project details

Repository contains all files for review and testing

4. Automation & AI

Workflow automation and AI-driven tasks using n8n

Automated network monitoring, alert handling, and response processes
